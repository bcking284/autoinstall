STEPS FOR FULL COMPUTER SETUP:
__________________________________________
USER GENERIC SETUP

1. *user* manually disable UAC
2. Join to domain and change name to user FLAST-MODEL
3. Grab NEWPCW10 from fileserver or have saved in folder with program
4. Change view to large icons
5. Install absolute
6. Install Access Panel Extension.msi
7. install MitelConnect.exe
8. (optional) Drag Howtovpn out to desktop
9. Install PrinterinstallerClientSetup.exe
10. Install adobe reader
11. Install Teams
12. Install unattended-updater
13. Install VPN
14. Go up to NEWPCW10
15. Open img-inst and run SETUP.BAT as admin
16. Install imaging and Kofax
17. Install Dell Display Manager

USER SPECIFIC SETUP
1. Log in as USER, prompt for username/password/...
2. start windows updates
3. start dell command updates
4. log in MitelConnect
5. log in VPN
6. Companyapps/devapps/(optional)Leasecomplete
7. Teams
8. Outlook
9. OneDrive
10. Chrome (homepage, printerlogic)
11. Set default Outlook, Adobe, Chrome
